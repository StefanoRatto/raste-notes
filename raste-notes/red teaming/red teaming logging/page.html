Install screen:
  • sudo apt install screen
Create and edit .screenrc:
  • defshell -bash
  • startup_message off
  • multiuser on
  • defscrollback 10000
  • logfile /home/$USER/screenlogs/$USER-screenlog.%H.%n.%Y%m%d-%0c:%s.%t.log
  • logfile flush 5
  • logtstamp on
  • deflog on
  • defmonitor on
  • caption always "%{= gk}%-Lw%{= bW}%50> %n%f* %t %{-}%+Lw%< %= %{= rk} %H %l %{= gk} %0c:%s %{-}"
  • defutf8 on
  • activity "Activity in %t(%n)"
Add to .bashrc:
  • PS1='[`date  +"%d-%b-%y %T"`]\[\033[01;31m\] `ifconfig ens33 2>/dev/null | sed -n 2,2p | cut -d" " -f 10`\[\033[00m\] \[\033[01;34m\]\W\[\033[00m\] > '
Create logs folder:
  • mkdir /home/$USER/screenlogs
Run screen:
  • screen
