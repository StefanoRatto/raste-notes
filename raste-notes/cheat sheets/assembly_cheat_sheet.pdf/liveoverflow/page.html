Simple Tools and Techniques for Reversing a binary - bin 0x06: https://www.youtube.com/watch?v=3NTXFUxcKPc
  • Show file info:
    • raste@raste-dev-01:~$ file "filename"
  • Show memory dump:
    • raste@raste-dev-01:~$ hexdump -C "filename"
  • Show strings in the binary:
    • raste@raste-dev-01:~$ strings "filename"
  • Decompiler, show the code:
    • raste@raste-dev-01:~$ objdump -d "filename"
  • Decompiler, show the headers:
    • raste@raste-dev-01:~$ objdump -x "filename"
  • Show syscalls:
    • raste@raste-dev-01:~$ strace "filename"
  • Show libraries used by the binary:
    • raste@raste-dev-01:~$ ltrace -d "filename"
